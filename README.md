# SwitchCat

SwitchCat is an experimental project that allows users to generate random cat images and download them. The project uses Vite and Node.js for efficient development.

## Project Overview

SwitchCat is a web application that leverages Vite and Node.js to provide users with a delightful experience of generating and downloading random cat images. The project is in an experimental stage, aiming to explore creative possibilities with cat imagery.

## Getting Started

To run SwitchCat locally, follow these steps:

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies with `npm install`.
4. Start the development server with `npm run dev`.
5. Open your browser and go to `http://localhost:3000`.

Feel free to explore and contribute to the project!
